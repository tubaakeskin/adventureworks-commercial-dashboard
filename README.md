# 📊 AdventureWorks Commercial Intelligence Suite

An interactive, multipage **Decision Support System (DSS)** built to empower Commercial, Sales, Finance, and Country Managers with actionable business insights, predictive forecasting, and strategic simulations.

## 🚀 Live Application
👉 [Click here to open the Live Dashboard](https://adventureworks-commercial-dashboard.streamlit.app/)

---

## 📌 Project Overview
This project transforms raw AdventureWorks transactional data into a centralized commercial cockpit. Rather than just displaying charts, the system actively diagnoses operational leaks (like high product return waves), evaluates volume-vs-margin product portfolios, forecasts revenue, and provides dynamic strategy recommendations.

### Key Features by Page:
1. **📊 Executive Summary:** High-level dynamic KPI tracking (Revenue, Gross Margin, Order Volumes, and Return Rates) combined with a pro-active **Quick Diagnostic Tool** that alerts managers to operational leaks.
2. **🛍️ Sales Performance:** Category performance tracking (Revenue vs. Net Profit), regional market shares, and top 10 performing customers and products.
3. **💸 Profitability & Returns:** A custom **Volume vs. Margin Matrix (Quadrant Analysis)** that categorizes products into "Problem Stars" or "Niche Champions", matched with a financial leakage audit of returns.
4. **🔮 Forecasting & Strategy:** Statistical 12-month linear regression extrapolations combined with an automated **Executive Recommendation Engine** that delivers dynamic strategic advice.
5. **🎛️ Scenario Simulation:** An interactive **"What-If" planning tool** allowing managers to slide levers for price changes, volume deltas, and return rate optimizations to preview the direct bottom-line impact.
